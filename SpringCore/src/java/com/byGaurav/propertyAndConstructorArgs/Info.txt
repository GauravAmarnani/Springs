The basic concept is that : 
1) Student class object will be created in Exam class using bean of spring. 
2) That Student class object will have its own different properties (3).
3) There will be 2 Student objects. 
4) Both of them will set their 3 properties (name , id , address) using spring.

   Second object using constructor-args feature.
